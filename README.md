# reb's portfolio

An interactive retro desktop portfolio built with React, TypeScript, Tailwind CSS, and Vite.

Instead of a standard scrolling resume page, this site opens like a small old-school desktop. Visitors can drag windows around, browse portfolio folders, read experience and education details, open mini tools, and play with a few interactive features.

## Features

- Retro desktop interface with draggable windows
- Portfolio Explorer with About, Experience, Education, Leadership, Projects, and Socials sections
- Paint 95-style drawing tool
- 8-bit synth with sound effects, notes, and a small sequencer
- Cat Flap mini game
- Start menu, taskbar, desktop shortcuts, and responsive window behavior

## Tech Stack

- React
- TypeScript
- Tailwind CSS
- Vite
- Framer Motion / Motion
- Web Audio API
- HTML Canvas

## Run Locally

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```

## Deployment

This project is ready to deploy on Cloudflare Pages, Vercel, Netlify, or any static hosting platform that supports Vite.

Recommended Cloudflare Pages settings:

- Build command: `npm run build`
- Build output directory: `dist`
- Framework preset: `Vite`
